## Description

Библиотека содержит базовые примитивы для создания чейнкодов.  

* BaseToken
* BaseContract

Библиотека реализует функциональность

* batching
* swapping
* multiswap

Библиотека реализует функциональность для взаимодействия с чейнкодом управления доступами(acl)
